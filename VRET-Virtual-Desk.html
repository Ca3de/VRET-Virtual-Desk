Virtual Destroy Process

Step-by-Step Workflow
  1. Move items from Destroy01-08 to green tote stickers (max 200/tote)
  2. Create PNH in Frost (FC Menu 237)
  3. Wait 3-5 minutes for system processing
  4. Pack out totes onto SPOO using station wsFracsLTL
  5. Get ship ID from simple pack tool or rodeo
  6. Manifest in Frost VMD with dstry-ltl method
  7. Use ind8-vrets@amazon.com for notifications

Stower Lost Cart Recovery

Container Recovery Process
Lost Cart Recovery Steps:
1. Go to FC Research (59) → Type pc99login (e.g., pc99dabechu)
2. Scroll down and copy SKU under Scannable ID
3. Go to Barcode Generator → Select Code 128
4. Paste Scannable ID (e.g., tsSTOW50222) → Generate Sheet
5. Use Move Container (1057) to relocate individually
